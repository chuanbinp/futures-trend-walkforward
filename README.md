# Futures Trend Walk-Forward

Final project for **MATH GR5360: Mathematical Methods in Financial Price Analysis** at Columbia University.

This project studies inefficiencies in futures markets using two Random Walk diagnostics—**Variance Ratio** and **Push-Response** tests—and evaluates a **channel-based trend-following trading system** under a **walk-forward optimization** framework. The analysis is performed on **PL futures** and one chosen secondary futures market using intraday OHLC data.

## Project Goals

- Understand the microstructure and behavior of the selected futures markets
- Test for market inefficiencies in each time series
- Identify whether inefficiencies are consistent with trend-following or mean-reversion
- Locate the approximate time scales where inefficiencies appear
- Implement the trend-following system
- Optimize strategy parameters in-sample and evaluate them strictly out-of-sample
- Compare in-sample and out-of-sample performance decay
